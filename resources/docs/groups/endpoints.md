> Example request:

```bash
curl -X POST \
    "http://localhost:8000/api/todo" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \
    -d '{"title":"facilis","note":"quia","complete_todo":false}'

```

```javascript
const url = new URL(
    "http://localhost:8000/api/todo"
);

let headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "title": "facilis",
    "note": "quia",
    "complete_todo": false
}

    method: "POST",
```
